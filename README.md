# victron-dbus-awattar

A Victron Gui Package for charging batteries based on (Awattar) market prices.

# Install
copy all files to /data/dbus-awattar on your cerbo gx

cd /data/dbus-awattar

./setup.sh INSTALL

# Uninstall
./setup.sh UNINSTALL
